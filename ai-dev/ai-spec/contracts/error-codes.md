# error-codes.md — SuperSonic 错误码规范

## 错误码结构
格式：`S2-{模块}-{序号}`

## 通用错误码

| 错误码 | HTTP Status | 说明 |
|--------|------------|------|
| S2-COMMON-001 | 400 | 参数校验失败 |
| S2-COMMON-002 | 401 | 未认证 |
| S2-COMMON-003 | 403 | 无权限 |
| S2-COMMON-004 | 404 | 资源不存在 |
| S2-COMMON-005 | 409 | 状态冲突（非法状态流转） |
| S2-COMMON-006 | 500 | 内部错误 |

## 语义层错误码

| 错误码 | 说明 |
|--------|------|
| S2-HEADLESS-001 | NL2SQL 解析失败 |
| S2-HEADLESS-002 | 模型未上线，无法查询 |
| S2-HEADLESS-003 | 数据集引用了未发布的模型 |
| S2-HEADLESS-004 | SQL 执行超时 |

## 对话层错误码

| 错误码 | 说明 |
|--------|------|
| S2-CHAT-001 | Agent 未启用 |
| S2-CHAT-002 | 查询超出 Agent 数据集范围 |
| S2-CHAT-003 | 多轮对话上下文丢失 |

## 认证层错误码

| 错误码 | 说明 |
|--------|------|
| S2-AUTH-001 | 租户不存在或已禁用 |
| S2-AUTH-002 | Token 过期 |
| S2-AUTH-003 | 跨租户访问拒绝 |

## 多租户错误码

| 错误码 | 说明 |
|--------|------|
| S2-TENANT-001 | 租户上下文缺失 |
| S2-TENANT-002 | 跨租户数据访问拒绝 |
