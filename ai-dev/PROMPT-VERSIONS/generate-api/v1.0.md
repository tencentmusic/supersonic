## 任务描述
生成：Controller + Service 接口 + ServiceImpl + DO + Mapper + DTO + 对应测试骨架

## 强制约束（SuperSonic 专项）
- 依赖注入：@RequiredArgsConstructor + private final，禁止 @Autowired
- REST 风格：Google RESTful API design（resource-oriented URLs）
- 跨模块通信：Spring ApplicationEvent，禁止 reflection
- 多租户：DO 类必须包含 tenantId 字段（排除表除外）
- MyBatis-Plus：使用 ServiceImpl 基类 + LambdaQueryWrapper
- SPI 注册：spring.factories

## 验收条件
- [ ] Controller 中不含任何 if/else 业务判断
- [ ] 所有入参有 @Valid + 对应约束注解
- [ ] 返回结构严格匹配统一 Response 格式
- [ ] 使用构造器注入（@RequiredArgsConstructor）
- [ ] DO 类包含 tenantId 字段
- [ ] 异常处理使用 @ControllerAdvice
- [ ] intent.md Anti-Goals 是否有对应测试用例？
- [ ] 复杂逻辑是否标注了 @AiGenerated？

## 输出格式
1. 《生成决策说明》（含推理链）
2. 代码文件（Controller / Service / ServiceImpl / DO / Mapper / DTO）
3. 测试骨架文件
